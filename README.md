# CheckSite - Automated Site Availability Monitor
A lightweight Python autmation utility that tracks the uptime of specific website (e.g google.com) at scheduled intervals and provides real-time Online / Offline status notifications.

## Features 
* **Uptime Tracking:** Ping target domains to check server responsiveness.
* **Scheduled Tasks:** Runs at defined time intervals automatically.
* **Instant Alerting:** Notifies the user immediately if a critical site goes offline.

## Tech Stack
* Python 3.x
* Requests / Time / Socket libraries

## How to Run
1. Run the script: 'python CheckSite.py'
2. Customize the target URLs inside the script to monitor any website.
